轻松实现LED电源的NFC参数设置及LED光衰减的自动补偿
简化LED电流设置，提高LED驱动器市场价值!
　 　　　　　　　
尊敬的客户，
在LED市场中，越来越多的LED电源提供近场通信（NFC）编程和恒定光通量输出（CLO）功能。NFC编程功能旨在取代高人力成本的“插入式电阻”电流设置方法，并提高整个价值链的灵活性。具有CLO功能的产品可以通过调整LED电流来补偿LED模块的光通量下降（老化效应）。针对市场需求，Infineon（英飞凌公司）的工程师开发了 NFC-PWM 系列产品，创新性的将智能逻辑功能集成到NFC芯片中，并使用PWM信号控制模拟驱动器，无需微控制器即可在模拟系统中实现NFC编程和CLO功能。
为大批量，成本敏感的中低端LED电源市场提供了最具成本优势的解决方案。
亮点：
NLM0011是市场上第一款具有集成恒流明输出（CLO）功能的NFC配置芯片。英飞凌NFC-PWM解决方案帮助客户轻松实现LED电源的NFC参数设置及LED光衰减的自动补偿。
NLM0010是不具有CLO的轻型版本。
两款IC都具有众多高级功能，例如工作时间计时器以及开/关计数器。芯片符合大多数主流模拟LED驱动器设计，无需对系统进行重新设计，就可以轻松取代“插入式电阻器”的设计方案。
